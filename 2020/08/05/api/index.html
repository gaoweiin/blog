自建的api

s.weii.ga短链接接口

接口地址：https://s.weii.ga/api.php

返回格式：json

请求方式：get

请求示例：https://s.weii.ga/api.php?url=https://hexo.weii.ga/

名称 必填 类型 说明
url 否 string 要缩短的地址
名称 类型 说明
error string 错误类型
longurl string 原链接
shorturl string 缩短后链接
stats string 短链接状态监控

狗屁不通生成器接口

接口地址：https://api.weii.ga/b

返回格式：json

请求方式：get

请求示例：https://api.weii.ga/b?topic=abc&length=200

名称 必填 类型 说明
topic 是 string 要生成文章的主题
length 否 int 要生成的长度(默认6000)
名称 类型 说明
error string 错误类型
主题 string 文章的主题
文章 string 文章主体

狗屁不通英文版生成器接口

接口地址：https://api.weii.ga/be

返回格式：json

请求方式：get

请求示例：https://api.weii.ga/be?topic=abc&length=200

名称 必填 类型 说明
topic 是 string 要生成文章的主题
length 否 int 要生成的长度(默认6000)
名称 类型 说明
error string 错误类型
主题 string 文章的主题
文章 string 文章主体

“智能”对话生成器接口

接口地址：https://api.weii.ga/q

返回格式：json

请求方式：get

请求示例：http://www.weinas.ga:14253/q?question=qq

名称 必填 类型 说明
question 否 string 问题
名称 类型 说明
error_code int 错误类型
question string 问题
answer string 答案
